# 🎰 AI Casino Manager

A machine learning-based system that predicts player behavior and optimizes bonus strategies to improve engagement and retention.

---

## 🚀 Features
- 🎯 Predicts player churn using Random Forest  
- 🎁 Suggests optimal bonus strategies  
- 📊 Tracks player activity and behavior  
- ⚡ Improves decision-making using data  

---

## 🛠️ Technologies Used
- Python 🐍  
- Machine Learning (Random Forest) 🤖  
- Flask  
- HTML, CSS, JavaScript 🌐  

---

## 📂 Project Structure
- `app.py` → Main backend file  
- `templates/` → HTML files  
- `static/` → CSS, JS files  
- `model/` → Trained ML model  
- `dataset/` → Training data
